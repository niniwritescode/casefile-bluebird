# 🕊️ Case File: The Bluebird Killer

A noir detective mystery web game built with Python (Flask), HTML, CSS and JavaScript.

## 🔍 About

You are **Inspector Marcus Hale**.  
Over 18 months, five wealthy businessmen have been murdered across the city.  
The killer leaves one word behind — **BLUEBIRD**.

Read the case file. Interrogate suspects. Review evidence.  
Then name the killer.

## 🎮 Features

- Noir detective aesthetic with dark UI and typewriter effects
- Rain and lightning animation on welcome screen
- Ambient rain sound
- Interactive suspect dossiers with AI-generated portraits
- Polaroid-style evidence board with crime scene photos
- Witness statements with background profiles
- Detective's journal for taking notes
- Clearance code login (hint: think about the case name)
- Full story reveal with moral ambiguity ending

## 🛠️ Built With

- Python 3 + Flask
- HTML5 + CSS3 + JavaScript
- Jinja2 templates
- AI-generated character portraits

## 🚀 Run Locally

```bash
git clone https://github.com/niniwritescode/casefile-bluebird.git
cd casefile-bluebird
pip install flask
python app.py
```

Then open `http://localhost:5000`
